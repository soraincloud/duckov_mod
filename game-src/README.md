# game-src/

本目录存放《逃离鸭科夫（Duckov）》解包/提取出的游戏文件，用于在必要时做“只读参考”。

## 用途

- 查阅游戏内现有类型、字段、资源结构与命名（例如用于反射/Harmony patch 时确认签名）。
- 对照游戏程序集（Managed DLL）以配置 Mod 工程引用、理解运行时行为。

## 注意事项

- 这里不是日常开发目录：尽量不要在此修改文件来“实现功能”，功能开发应放在 `mods/`。
- 重要提示：解包文件对应的游戏版本**可能不是最新版本**；其中的类型/字段/资源结构可能已发生变化，因此只能作为参考对照，遇到差异以你本机当前游戏版本为准。
- 建议把它当作只读素材/证据库：当需要确认某个类/资源是否存在、具体路径/命名是什么时再来查。
- 若你计划将本仓库公开或同步到远端，请自行确认：解包文件是否适合纳入版本控制、是否会导致仓库体积过大或触及发行/版权方面的限制。
