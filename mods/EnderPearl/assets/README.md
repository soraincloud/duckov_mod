# EnderPearl 资源目录（assets）

这个目录用于存放“源码仓库里的原始素材”，例如：

- `item-icons/`：物品图标/贴图（建议 png）
- `bundles/models/`：模型相关 AssetBundle（例如 `enderpearl_assets`）
- `bundles/sfx/`：音效相关 AssetBundle（预留）

说明：
- `bundles/` 下的文件是已打包的二进制资源（AssetBundle/类似产物），不要把原始素材直接塞进这些文件里。
- MOD 根目录的 `preview.png` 是游戏识别 MOD / Mods 菜单预览所需文件之一，建议保留在 MOD 根目录。
